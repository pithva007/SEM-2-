# Sign-In Form Project

This project is a simple sign-in form that includes various fields for user input. The form is designed to collect essential information from users, including their address, city, state, date of birth, image file upload, email, phone number, pin code, password, and confirm password.

## Project Structure

The project consists of the following files:

- **index.html**: Contains the structure of the sign-in form with all necessary input fields and buttons.
- **css/styles.css**: Contains styles for the sign-in form, enhancing the visual presentation and user experience.
- **js/script.js**: Contains JavaScript code for form validation, ensuring that all required fields are filled out correctly before submission.

## Getting Started

To set up and run the project, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd sign-in-form-project
   ```

3. **Open the `index.html` file** in your web browser to view the sign-in form.

## Usage

- Fill in all required fields in the form.
- Click on "Verify and Sign Up" to submit the form.
- Use the "Reset Form" button to clear all fields.
